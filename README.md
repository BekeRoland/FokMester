FokMester is a multilingual Flutter application for alcohol strength
temperature correction, dilution calculations, and liquid mixing.

The application was developed as a portfolio project with a focus on
calculation accuracy, input validation, localization, accessibility,
persistent local data, and automated testing.

## Features

- Alcohol strength correction to 20 °C
- Two-dimensional interpolation between table values
- Dilution calculation for an existing spirit
- Required spirit and water calculation for a desired final volume
- Mixing calculation for two liquids
- Liter, deciliter, and milliliter support
- Hungarian, English, and Romanian localization
- Light, dark, and system themes
- Locally stored calculation history
- Responsive Material 3 interface
- Unit and widget tests
